# DWZ
这是一个短网址程序
主要实现：短网址生成，页面跳转，短链访问信息统计，跳转页面渲染。带有用户登陆模块。
<br>
数据库mongodb使用findandmodify进行id自增。
这是一个简单的短链程序。
